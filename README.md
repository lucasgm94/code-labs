# 🚀 code-labs

### 🧠 Objective
A personal sandbox for hands-on practice with **SQL**, **PL/SQL**, and **Python**. This repository stores exercises, small projects, and code snippets developed while learning and strengthening programming skills.

---

## 🛠️ What's Inside?

### 🗄️ SQL & PL/SQL (Oracle APEX)
* **Querying:** Practice with SELECTs, JOINs, subqueries, and views.
* **Database Programming:** Writing stored procedures and basic PL/SQL logic.
* **Oracle APEX Labs:** Building functional dashboards and interfaces from scratch.
* **Data Management:** Working with table structures (DDL) and data manipulation (DML).

### 🐍 Python
* **Scripting:** Small scripts for automation and logic exercises.
* **Database Connection:** Basic exercises connecting Python with SQL databases.
* **General Practice:** Algorithmic thinking and data manipulation tasks.

---

## 📂 Folder Structure
* **`/PL-SQL`**: Advanced database labs (e.g., `crypto-analytics-dashboard`).
* **`/SQL`**: Pure SQL query exercises and challenges.
* **`/Python`**: Python scripts and practice files.
