# 🐍 Python Development Labs

### 🧠 Objective
A personal environment for mastering **Python** through continuous practice. This folder tracks my progress from fundamental logic and syntax to the development of modular scripts and functional applications.

---

## 🛠️ Development Environment
* **Language:** Python 3.x
* **IDE:** PyCharm (Professional/Community)
* **Focus:** Logic building, script automation, and clean coding standards.

---

## 📂 Laboratory Organization
The repository is structured to separate learning modules from practical implementations:

* **`/classes`**: Structured lessons and exercises (organized by `day-01`, `day-02`, etc.).
* **`/projects`**: Small, self-contained Python applications and logic challenges.

---

## 🚀 Skills & Concepts Practiced
* **Foundations:** Variable management, data types, and string manipulation.
* **Logic & Flow:** Implementing mathematical operators, rounding, and control structures.
* **Automation:** Writing scripts to handle repetitive tasks or data processing.
* **Modular Code:** Using PyCharm's features to write organized and PEP 8 compliant code.
