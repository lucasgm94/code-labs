# 🗄️ SQL Practice & Projects

### 📚 Source & Attribution
The database schemas and datasets in this folder are primarily based on exercises from the course:
**"SQL TOTAL - Domina Bases de Datos de 0 a Avanzado en 12 Días"** on Udemy. These resources are used strictly for learning and hands-on practice purposes.

---

## 🛠️ Current Environment
* **Tools:** MySQL Workbench, pgAdmin 4, SQL Server Management Studio 22
* **Cloud Infrastructure:** AWS RDS (Amazon Relational Database Service)
* **Status:** Actively practicing query optimization and schema management.

## 📁 Folder Structure
* **`/aws-rds-conection`**: Configuration and scripts for cloud database integration.
* **`/schemas`**: Table definitions and database architecture exercises.
* **`/classes`**: Structured exercises from the learning modules.
* **`/projects`**: Specific SQL implementations and small case studies.
* **`/screenshots`**: Visual evidence of query results and workbench setups.

## 🚀 Skills Practiced
* **DML:** Advanced SELECTs, JOINs, and complex data manipulation.
* **DDL:** Designing and modifying table structures.
* **Connectivity:** Integrating local environments with cloud-hosted (AWS) databases.
